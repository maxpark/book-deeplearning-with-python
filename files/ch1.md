
### 머신러닝 기초
```
 - 인공지능 > 머신러닝 > 딥러닝

 - 딥러닝 : 데이터의 반복 학습을 통하여 데이터의 숨어있는 패턴을 찾기
      (기존 프로그램 방식이 데이터의 패턴을 프로그래밍 하는 형태)
```

### 머신 러닝  학습 방식 
```
 - 지도 학습
   + 입력 데이터와 정답 라벨의 답의 쌍으로  학습
   + 데이터셋 준비 어려워

 - 비지도 학습
   + 정답 라벨 없이  데이터 규칙성 찾기
   + 활용예) 추천상품 찾아내기, 자연어 처리 등

 - 강화 학습
   + 에이전트의 행동에 대하여 환경이 보상/평가해 주고, 보상에 따라 다음의 에이전트 행동을 결정하는 형태
        에이전트 --행동--> 환경
        에이전트 <--보상-- 환경
   + 요즘 많이 연구한다는데.....
```
